# 显示转换
## 1.字符串转数字
```js
var x = '123';

console.log(~~x); // 123
console.log(+x); // 123
console.log(Number(x)); // 123
console.log(JSON.parse(x)); // 123
```
## 2.浮点数转整型
```js
var x = 8.24;

console.log(~~x); // 8
console.log(x | 0); // 8
console.log(x ^ 0); // 8
console.log(~~x); // 8
console.log(x << 0); // 8
console.log(x >> 0); // 8
console.log(parseInt(x)); // 8
console.log(Math.floor(x)); // 8
console.log(Math.round(x)); // 8
console.log(Math.ceil(x)); // 9
```
